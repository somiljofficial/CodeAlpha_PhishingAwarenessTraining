# CodeAlpha_PhishingAwarenessTraining

## 📌 Project Overview
This project is submitted as part of the **CodeAlpha Cyber Security Internship**.
It is a Phishing Awareness Training module designed to educate users on how to
recognize phishing emails, fake websites, and social engineering tactics, along
with an interactive quiz to reinforce learning.

## 🎯 Objectives
- Explain what phishing is and why it works
- Teach recognition of phishing emails and fake/spoofed websites
- Explain common social engineering tactics used by attackers
- Provide best practices to avoid falling victim
- Test understanding via an interactive quiz

## 📂 Repository Contents
| File | Description |
|------|-------------|
| `PHISHING_AWARENESS_TRAINING.md` | Full training content (use as slide/module script) |
| `quiz.html` | Standalone interactive quiz (open in any browser) |
| `README.md` | This file |

## 🧠 How to Use
1. Read/present `PHISHING_AWARENESS_TRAINING.md` as your training module or convert it into slides.
2. Open `quiz.html` in a browser to test knowledge interactively.
3. Record a short video walkthrough and post it on LinkedIn tagging **@CodeAlpha**, along with this repo link.
